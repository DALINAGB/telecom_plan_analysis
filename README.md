# 📊 Telecom Tariff Analysis: Megaline Profitability Optimization

## 📝 Business Overview
Este proyecto realiza un análisis estadístico profundo sobre el comportamiento de consumo de 500 clientes de la empresa **Megaline**. El objetivo principal es identificar cuál de los planes de prepago (**Surf** vs. **Ultimate**) genera mayor rentabilidad para orientar de manera eficiente el presupuesto de marketing de la compañía hacia el modelo de negocio más sólido.

## 🎯 Project Objectives
* **User Profiling:** Analizar patrones de consumo de minutos, mensajes y datos por plan tarifario.
* **Statistical Comparison:** Evaluar las diferencias en el ingreso promedio mensual entre planes.
* **Hypothesis Testing:** Validar estadísticamente si las variaciones de ingresos son significativas o producto del azar.
* **Strategic Consulting:** Emitir recomendaciones basadas en datos para la toma de decisiones ejecutivas.

## 📈 Key Insights & Results

### 1. Profitability Analysis
Tras realizar las pruebas de hipótesis (T-test), los resultados arrojan una diferencia estadísticamente significativa en los ingresos:
* **Plan Ultimate:** Ingreso promedio de **$72.31** (Mayor estabilidad y predictibilidad).
* **Plan Surf:** Ingreso promedio de **$60.71** (Alta volatilidad debido a cargos por excedentes).

### 2. Regional Consistency
Se realizó una comparación entre la región de **NY-NJ** y el resto de las locaciones. No se encontró evidencia estadística suficiente para afirmar que el comportamiento de gasto varía según la región, lo que sugiere que la estrategia de precios puede ser uniforme a nivel nacional.

### 3. Data Integrity
* Se procesaron datos de 500 usuarios del año 2018.
* Se realizó una limpieza exhaustiva de valores ausentes y corrección de tipos de datos para asegurar la precisión del análisis estadístico.

## 💡 Strategic Recommendations
1. **Focus on Ultimate:** Priorizar las campañas publicitarias hacia el Plan Ultimate debido a su mayor margen de beneficio y consistencia de flujo de caja.
2. **Upselling Strategy:** Implementar notificaciones para usuarios del Plan Surf con altos excedentes, invitándolos a migrar al Plan Ultimate, mejorando así la retención y el LTV (Lifetime Value).
3. **Regional Marketing:** Dado que no hay diferencias regionales significativas, se recomienda una campaña de marketing centralizada para reducir costos operativos.
